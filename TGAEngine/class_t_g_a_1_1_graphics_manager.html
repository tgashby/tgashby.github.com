      init

      Initializes the window to a specific size, title, and position.

All parameters are optional. Default values are: 1024x512, no title, and the window centered in the screen.

Parameters:
screenWidth- the window width
screenHeight- the window height
winTitle- the title of the window (what appears in its upper-left corner)
screenX- the x position of the window on the screen
screenY- the y position of the window on the screen
